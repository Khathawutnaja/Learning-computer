%d ข้อมูลตัวเลขจำนวนเต็ม
%f ข้อมูลตัวเลขที่มีจุดทศนิยม
%c  ข้อมูลตัวอักษร
%s ข้อมูลข้อความ มีลักษณะเป็นชุดของตัวอักษร ตัวเลข หรือ อักขระพิเศษ อยู่ในพื้นที่เครื่องหมาย ""
\n ขึ้นบรรทัดใหม่
\t เว้นช่องว่างในแนวนอน
----------- DATA TYPE -----------
Boolean ค่าทางตรรกศาสตร์ (Tru =1 false =0)
int  ตัววเลขไม่มีทศ
Float ตัวเลขมีทศ
char ตัวอักษร
string กลุ่มตัวอักษร
